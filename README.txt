SJNC Kites Absence Planner — Fixed Version

Open index.html in a web browser.

This version has:
- All dates and rounds permanently built into the page
- A visible Absent text box for every playing round
- Space for unlimited names, one per line
- Automatic local saving
- Save, export, clear and print buttons

Note: entries are saved only on the device/browser currently being used. A shared version for multiple families requires online hosting and a shared database.
